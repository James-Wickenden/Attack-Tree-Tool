Help

Loading into an editor

To open an editor, select one of the three options on the left [shown below], then click the Start Editing button on the right.

To create a group, choose a group code before clicking the start editing button. If that code is free, you'll create a live group editing session that anyone with that code can join.

To join a group, enter the code of the group you want to join before clicking the start editing button. If that code is valid, you'll join the live group editing session with equal editing access to all other members.

Using the editor

Once inside the editor, you can interact with two main windows. The left window is for graph editors, and contains two different tabs for viewing and editing the graph in two different ways.

The right window is for the attribute editor, and allows you to add, remove, see, and edit attributes in the attack tree.

Left window

The Graph View tab renders a visual tree. You can drag the tree around, and click nodes to select them and view their attributes in the attribute editor.

You can right click nodes to add children to them, toggle them between AND and OR nodes, and delete them and their children, where valid.

The List View tab displays the tree in a list format. You can select nodes, and perform the same functions as by right clicking nodes in the graph view. Editing this view also will update the graph view, and visa versa.

Right window

The Attribute Editor contains three different sub-menus: the Attribute List, the Attribute Creator, and the Cell Attribute Value Editor.


The Attribute List displays a list of all the attributes currently associated with that tree. Each node in the tree will have an associated value for that attribute, which is calculated according to that attributes set of rules. These rules are defined by that attributes domain.

You can click to toggle each one independently, which will change if its value is rendered for each node in your graph view. You can also delete attributes with the delete button, which will ask for a prompt to confirm this action. Finally, you can click New Attribute to open the attribute creator below.

The Attribute Creator is a form which creates new attributes. A description is not required, although if given one it will be shown when you hover over attributes in the attribute list. The domain controls what values the attribute can take, and the AND/OR rules for combining its values up the tree. The default value can only be one valid within that attributes domain.

The Cell Attribute Value Editor is a form which is context sensitive to the cell just clicked in the graph view. Here, you can view that node's attribute values, and change them if that node is a leaf in the tree.

Live collaborative editing

If in a group, any changes you make to the tree will be broadcast to every member of your group, and they will have their trees updated with the changes you've just made to yours. This may lead to some forms closing or refreshing. At the top of the screen, your group code is displayed, or there will be a line showing that you're working in a private session.

Options

The options dropdown shows a list of options you can perform.

  • The Refresh Tree option shouldn't be necessary for usage, but you can use it in a group to request a fresh version of the tree from the server, or in a private session to refresh the editors.
  • The Download Tree option saves a JSON file of the tree and its attributes to your computer.
  • The Upload Tree option prompts you for a JSON file in the same format as can be downloaded in the option above. When given, it will completely update the tree and its attributes to the new version, including sending it to your group members.
  • The help option brings you to this page.
  • The about option opens a page telling you about this project.
